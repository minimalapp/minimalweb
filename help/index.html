Primeros Pasos con Minimal

1. Registrar tu Cuenta

  1. Visita nuestra página de contacto y solicita acceso
  2. Recibe tu clave de API y credenciales de acceso por email
  3. Accede a tu dashboard personalizado

2. Configuración Básica

La integración más simple requiere solo una línea de código:

<script src="https://cdn.minimal.ai/v1/minimal.js" data-api-key="TU_API_KEY"></script>

3. Personalización Inicial

  • Colores Automáticos: Minimal detecta automáticamente los colores de tu marca
  • Posicionamiento: El toggle se posiciona automáticamente en la parte media inferior
  • Contenido: El sistema indexa tu contenido web automáticamente

4. Verificar la Instalación

Después de instalar el código:

  1. Refresca tu sitio web
  2. Busca el ícono de chat en la esquina inferior derecha
  3. Haz clic y prueba hacer una pregunta sobre tu contenido
  4. Verifica en tu dashboard que las interacciones se registren

Personalización Avanzada

Configuración Mediante Atributos

Puedes personalizar Minimal usando atributos en el script tag:

<script src="https://cdn.minimal.ai/v1/minimal.js" data-api-key="TU_API_KEY" data-position="bottom-left" data-primary-color="#2196F3" data-welcome-message="¡Hola! ¿En qué puedo ayudarte?"> </script>

Opciones de Personalización

  • data-position: "bottom-right" | "bottom-left" | "top-right" | "top-left"
  • data-primary-color: Color principal en formato hexadecimal
  • data-welcome-message: Mensaje inicial personalizado
  • data-placeholder: Texto del campo de entrada
  • data-theme: "light" | "dark" | "auto"

Configuración Avanzada con JavaScript

window.MinimalConfig = { apiKey: 'TU_API_KEY', theme: { primaryColor: '#2196F3', secondaryColor: '#3F51B5', borderRadius: '12px' }, behavior: { autoOpen: false, showTypingIndicator: true, enableSuggestions: true }, content: { welcomeMessage: '¡Bienvenido! ¿Cómo puedo ayudarte hoy?', placeholder: 'Escribe tu pregunta aquí...', errorMessage: 'Lo siento, hubo un error. Inténtalo de nuevo.' } };

Métricas Principales

  • Conversaciones Totales: Número total de sesiones de chat iniciadas
  • Tasa de Engagement: Porcentaje de visitantes que interactúan con el chat
  • Tiempo Promedio por Sesión: Duración media de las conversaciones
  • Tasa de Resolución: Porcentaje de consultas resueltas satisfactoriamente

Insights de Contenido

  • Preguntas Más Frecuentes: Identifica qué información buscan tus usuarios

  • Términos de Búsqueda: Palabras clave más buscadas

Acciones Recomendadas

  • Crea contenido específico para preguntas frecuentes sin respuesta
  • Optimiza páginas con alta demanda conversacional
  • Ajusta el mensaje de bienvenida según patrones de uso
  • Configura respuestas automáticas para consultas comunes

Preguntas Frecuentes

¿Cómo afecta Minimal a la velocidad de carga de mi sitio? +
Minimal está optimizado para carga asíncrona y pesa menos de 50KB. Se carga después del contenido principal de tu sitio, por lo que no afecta la velocidad de carga percibida. Además, utilizamos CDN global para garantizar tiempos de carga mínimos.
¿Funciona con mi plataforma actual (WordPress, Shopify, etc.)? +
Sí, Minimal es compatible con cualquier plataforma web. Solo necesitas poder agregar código JavaScript personalizado, lo que es posible en WordPress, Shopify, Squarespace, Wix, sitios estáticos, React, Vue, Angular y cualquier otra tecnología web.
¿Cómo maneja Minimal el contenido de mi sitio? +
Minimal indexa automáticamente el contenido público de tu sitio web. Respeta los archivos robots.txt y no indexa contenido privado o protegido por contraseña. El contenido se procesa de forma segura y se utiliza únicamente para responder consultas sobre tu sitio.
¿Puedo personalizar las respuestas del chat? +
Sí, puedes configurar respuestas personalizadas para preguntas específicas, definir el tono de comunicación, establecer reglas de negocio personalizadas y crear flujos conversacionales específicos para tu industria a través del dashboard de administración.
